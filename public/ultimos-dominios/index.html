Registro de dominios .cl en Chile: datos en tiempo real

Chile cuenta con más de 600.000 dominios .cl activos, administrados por NIC Chile, el organismo dependiente de la Universidad de Chile que gestiona el registro de nombres de dominio del país desde 1987. En promedio, se registran entre 300 y 600 nuevos dominios .cl cada día hábil, con picos de actividad los lunes y martes.

Esta página monitorea en tiempo real los últimos dominios inscritos en NIC Chile, actualizándose automáticamente cada hora mediante un sistema de scraping que procesa el registro público de nic.cl. Los datos están disponibles como Open Data en formato JSON y RSS para investigadores, desarrolladores y analistas del ecosistema digital chileno.

Fuente: Datos obtenidos del registro público de NIC Chile (nic.cl). Última actualización: 11-06-2026 16:47. Frecuencia: cada hora vía GitHub Actions.

Preguntas frecuentes sobre dominios .cl

1¿Cuántos dominios .cl se registran por día en Chile?

En promedio se registran entre 300 y 600 dominios .cl nuevos cada día hábil en Chile, según datos públicos de NIC Chile. Los lunes y martes suelen ser los días con mayor actividad de registro.

2¿Cómo puedo saber si un dominio .cl está disponible?

Puedes verificar la disponibilidad de un dominio .cl directamente en NIC Chile (nic.cl) o usando nuestra herramienta de análisis WHOIS. Si el dominio no aparece en el registro de NIC, está disponible para ser inscrito.

3¿Cuánto cuesta registrar un dominio .cl?

El costo de inscripción de un dominio .cl en NIC Chile es de aproximadamente $15.900 CLP + IVA por dos años. Algunos proveedores de hosting incluyen el dominio gratis al contratar un plan de alojamiento.

4¿De dónde provienen los datos de esta página?

Los datos se obtienen directamente del registro público de NIC Chile (nic.cl), el organismo oficial que administra los dominios .cl. Nuestro sistema automatizado actualiza la información cada hora mediante GitHub Actions.

5¿Puedo recibir alertas de nuevos dominios registrados?

Sí, puedes suscribirte a nuestro feed RSS en formato XML o JSON para recibir notificaciones automáticas de nuevos dominios .cl registrados. Los enlaces están disponibles en esta página.
